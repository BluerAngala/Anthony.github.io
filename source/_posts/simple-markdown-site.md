> 这可能是史上最简单的静态站点生成器了,使用markdown作为源文件,生成html静态站点


## 只有3个文件夹

1. posts:`存放markdown文件`

2. template : `模板文件`

3. public : `html发布文件`

## 使用方法:

```
git clone https://github.com/jingtaiboke/simple-markdown-site
cd simple-markdown-site
npm i
npm run build

```
